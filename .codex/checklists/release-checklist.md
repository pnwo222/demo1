# 通用发布检查表

- [ ] 前端构建通过。
- [ ] 后端或服务端构建通过。
- [ ] CI 全部通过。
- [ ] Coverage 未低于主分支基线。
- [ ] 安全扫描无高危。
- [ ] SAST / SCA / Secret Scan / DAST 已通过或有风险接受记录。
- [ ] 数据库 migration 已在预发验证。
- [ ] 数据库回滚或兼容策略已确认。
- [ ] 预发环境 UAT 通过。
- [ ] Feature Flag 已配置。
- [ ] 灰度范围已确认。
- [ ] Canary 观察窗口已确认。
- [ ] 全量发布条件已确认。
- [ ] 监控指标已配置。
- [ ] 告警规则已配置。
- [ ] 回滚方案已确认。
- [ ] 发布负责人已确认。
- [ ] 发布说明已准备。

## 重点观察指标

- 核心流程成功率。
- 核心流程失败率。
- 接口错误率。
- 接口 P95/P99 耗时。
- 数据库慢查询。
- 前端白屏率或 JS 错误率。
- 高风险业务事件数量。
- 用户反馈和异常工单数量。
