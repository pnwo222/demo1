# 纺院高校一卡通 H5 需求到原型页面蓝图

## 蓝图说明

- 需求来源：`docs/requirements/纺院需求.md` 中 H5-001 至 H5-013。
- 框架来源：`project/h5/`，主视口 375px，验证 320px、390px、414px。
- 认证策略：原型默认无认证；宿主 APP、Token、身份来源、服务消息与卡包协议均标记待确认。
- 数据策略：使用可切换的本地 mock 展示加载、正常、空、失败、禁用和成功状态。
- 标注策略：顶部栏与底部导航为全局作用域；业务内容和浮层按 `page:<page-id>` 隔离。
