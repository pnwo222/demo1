# Snowy 后管原型 Demo 框架

本目录用于后续原型阶段复用。它不是业务代码，也不需要构建。

## 使用方式

直接用浏览器打开：

```text
docs/design/prototype-demo-framework/index.html
```

## 依赖

运行时通过 CDN 引入：

- Vue 3.5.13
- Ant Design Vue 4.2.6
- dayjs 1.11.19

版本参考 `project/snowy-admin-web/package.json`，只保留原型必需依赖。工具图标使用页面内置字符图标，不额外依赖图标包。

## 原型规则

- 布局参考 Snowy 后管：左侧菜单、顶部栏、面包屑、页签、查询区、工具栏、表格、分页、抽屉和弹窗。
- 标题、Logo、系统名称必须根据具体需求、标书或项目定义配置；不要沿用 Demo 默认值。
- 页面只放业务用户能理解的内容，不展示开发实现提示、组件名、内部变量、接口/mock 说明或权限函数。
- 所有看起来可点击的元素都必须能点击，并给出对应反馈或状态变化。
- 表格内容要符合表头语义：图片字段展示图片或空图占位，状态字段使用标签或开关，时间字段展示日期时间，排序字段展示数字。
- 图片上传参考 Snowy 图片卡片上传形态：支持选择本地图片、回显缩略图、预览和移除。
- 常见展示形态已预设：封面图、头像、文件类型、标签、徽标、开关、进度、长文本省略、树形选择、更多操作、确认弹窗、导入拖拽上传。
- 后续业务原型可以复制本目录，替换菜单、字段、数据和流程。
